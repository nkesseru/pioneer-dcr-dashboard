# 02 — Cleaning Operations

The customer's business. Operators who skip this become forgettable vendors who happen to sell to cleaning companies.

Maps to **Pillar 02** in the [Competency Model](../00-academy-overview/02-competency-model.md).

## Competencies covered

| ID | Competency | Target level by graduation |
|---|---|---|
| 02.1 | Service models (recurring, day porter, specialty, supply) | 2 |
| 02.2 | Labor economics (RPLH, productivity, scheduling) | 3 |
| 02.3 | Quality systems (inspections, DCRs, feedback loops) | 2 |
| 02.4 | Billing patterns (recurring, AR aging, revenue base) | 2 |
| 02.5 | Operational pain points (callouts, supplies, complaints) | 2 |

## Mandatory experience

**One full overnight cleaning shift before completing Pillar 02.** No exceptions. The operator cleans alongside a Pioneer tech, sees the work, takes it home in their body.

This is non-negotiable. If a candidate refuses, end the hiring process.
