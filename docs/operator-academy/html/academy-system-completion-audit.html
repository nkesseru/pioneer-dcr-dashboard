6. Launch decision

Recommendation: Ready with exceptions.

What "Ready" means here

The Academy can run Cycle 1 end-to-end without tribal knowledge:

  • ✅ A candidate can find the entry point (founder-pilot-start-here.md or the founder-pilot-html.html UI)
  • ✅ A reviewer can read their job in 30 minutes (onboarding kit)
  • ✅ Every evidence artifact has a template
  • ✅ Every grading instrument has a rubric
  • ✅ Every cycle phase has a workflow (submission → intake → scorecard → calibration → decision → Lineage ID → letter → cycle record → archive)
  • ✅ The decision rule is explicit (worst-of-three Bootstrap; framework-defined post-Bootstrap)
  • ✅ Edge cases have fallback paths (Day 1 fallback, Conditional Pass remediation, Not Yet re-test, Public Review)

What "with exceptions" means

The 3 Important gaps and 8 Nice to Have gaps remain. None block Cycle 1. They are:

  • Quality-of-life improvements that close as Cycle 1 patterns surface them
  • Stylistic standardizations (trainer paragraph, business outcome doc) that don't change what reviewers grade
  • Implicit workflows (weekly debrief) that the Constitution covers but doesn't structure
