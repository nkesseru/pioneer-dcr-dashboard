Vital Read Recording Standard

This is the single source of truth for how a Vital Read is recorded for certification evidence. If a recording doesn't meet this standard, it doesn't count.

The standard

Item Specification
Tool Loom (default) or any tool producing a sharable mp4/screen-recording with audio. No Zoom cloud recordings — too easy to lose, too hard for reviewers to access.
Length 15–25 minutes. Hard cap at 30. A Vital Read longer than 25 min usually means the operator is presenting, not reading.
Audio Required. Both sides audible. If the customer is on the call, their consent to record must be captured in the first 30 seconds (verbal "yes" on tape).
Screen PioneerOps visible for the majority of the read. The Six Vitals on screen for at least one named pass.
Naming vital-read_<account-shorthand>_<YYYY-MM-DD>_<candidate-initial>.mp4 (e.g., vital-read_acme-cleaning_2026-06-20_a.mp4)
Storage Single shared folder for the cycle. One folder per candidate. Reviewer-readable link. No private Loom accounts the reviewer can't view.

What disqualifies a recording

  • No audio, or one-sided audio (only the operator talking, customer muted)
  • The Six Vitals never shown on screen
  • The operator delivering a monologue with no customer interaction
  • Recording starts after the read began (must capture full session including opening)
  • File link the reviewer cannot open
  • File name that doesn't follow the convention (reviewers index by name)

Recording the 8 reads — practical sequence

Week Recording
1 2 recordings (1 per owned account)
2 2 recordings
3 2 recordings
4 2 recordings
Total 8 recordings — meets Operator I framework §2 bar

The 4 consecutive weeks requirement is not negotiable. A 4-recording week followed by a 4-recording-three-weeks-later does not count.

What the reviewer is looking for

  • Did the candidate let the customer read, or did they take the read away?
  • Did the candidate name what each vital actually measured, or recite labels?
  • When a vital crossed a threshold, did the candidate name the pattern from the Pattern Library or improvise?
  • Did the candidate end with a named verification step (the start of a Loop entry)?

These are the cuts the reviewer makes. Recordings that pass production quality but fail substance quality still don't count.
