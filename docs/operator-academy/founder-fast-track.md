## What about the 4-week Vital Read floor?

The single hardest question for the Founder Pilot fast track.

The framework §2 requires "8 reads total, recorded · 4 consecutive weeks." This is doctrine. Nick cannot bypass it without modifying doctrine (which the user explicitly forbade).

**Possible compressions considered, all rejected:**

| Compression idea | Why rejected |
|---|---|
| Do 8 reads in 2 weeks (4 per week) | Violates "4 consecutive weeks" framework bar |
| Daily reads for 8 days | Violates "weekly" cadence doctrine in framework + Constitution |
| Skip Vital Reads entirely because Nick "already" does this | Violates Operator I bar; would set a precedent April couldn't follow |
| Use archival recordings Nick has from past customer interactions | Not produced under the Vital Read recording standard; would fail the recording standard's bar |
| Reduce from 8 to 6 reads | Modifies framework §2 |

**The 4-week floor stays.** Founder Pilot certification cannot complete before Week 4. Decision Friday 2026-07-24 stands.
