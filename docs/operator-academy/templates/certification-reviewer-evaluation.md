# Reviewer Evaluation

Used at **Step 3** of the certification process. Each of the two reviewers independently completes this form. Their ratings + defenses feed the Academy lead's Step 4 decision.

The framework requires **independence**: reviewers must complete their evaluation before seeing the other reviewer's. The Academy lead enforces this.

If two reviewers' ratings differ by more than 1 point on any requirement section, the Academy lead convenes them for a reconciliation conversation before deciding.

## Scoring Scale

Every requirement section is rated 0–4:

| Score | Meaning |
|---|---|
| 0 | Did not demonstrate |
| 1 | Demonstrated with significant gaps |
| 2 | Demonstrated competently |
| 3 | Demonstrated at the level expected of senior operators |
| 4 | Demonstrated exceptionally — sets a new bar for the team |

For each rating, the reviewer writes a one-paragraph defense. **Empty defenses invalidate the rating.**

| Used "dashboard" when the doctrine says "diagnostic instrument" | Yes / No | |
| Curated their own evidence record (only-wins) | Yes / No | |

**Any Yes triggers immediate Academy lead review.** A "Must Not" violation is grounds for either No Pass — Disqualifying OR Conditional Pass with explicit remediation.

## Overall Recommendation

The reviewer writes their overall recommendation. Specific. Defensible. No hedging.

| Recommendation | When |
|---|---|
| **Strong Pass** | All sections ≥ 3, Pioneer Test pass, no Must Not violations |
| **Pass** | All sections ≥ 2, Pioneer Test pass, no Must Not violations |
| **Conditional Pass** | 6 of 8 sections ≥ 2, Pioneer Test pass, one specific gap |
| **No Pass — Gap-Driven** | 2+ sections below 2, OR Pioneer Test fail |
| **No Pass — Disqualifying** | Any Must Not violation, OR only-wins evidence record |
