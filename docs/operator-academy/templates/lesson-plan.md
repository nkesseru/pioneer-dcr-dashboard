If a section feels hard to fill in, that is the signal you do not understand the topic well enough to teach it yet.

## ELEMENT 11 — Coaching Discussion

What the trainer asks at the Friday debrief to find out whether the lesson actually landed.

Three to five questions. Open-ended. Not leading.

> 1. What surprised you in this lesson?
> 2. Where did you almost get it wrong on the exercise?
> 3. Which of your owned accounts could this apply to next week?
> 4. What part is still unclear?
> 5. What would make this lesson sharper for the next operator?

## PUBLISH GATE — The Pioneer Test

Every lesson must pass these checks **before** publishing. If any box is unchecked, the lesson is not ready.

- [ ] **Would Nick teach it this way?** If no, rewrite.
- [ ] **Does it teach an outcome, not a feature?** If feature talk, rewrite.
- [ ] **Is the Action → Metric → Outcome triple filled in?** If not, stop.
- [ ] **All 11 elements present?** Every section filled. No "TBD."
- [ ] **Reading level 8th–10th grade?** Read it aloud. Short sentences. No MBA jargon.
- [ ] **Pioneer + customer examples are different?** Both must be real.
- [ ] **Visual diagram present?** No diagram → concept is tangled.
- [ ] **Exercise has real data + a real output?** No theoretical exercises.
- [ ] **Role-play scenario named?** Where will the operator use this under pressure?
- [ ] **Vocabulary added to the Operator Glossary?** Every new term, in the glossary.
- [ ] **Skimmable?** Bullets, tables, callouts, examples. Not paragraphs.

If all 11 boxes are checked: ready to teach.

If any box is unchecked: rewrite or remove.
