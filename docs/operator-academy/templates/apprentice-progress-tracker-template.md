## 17. Certification Gate Checklist

### Operator I Gate

#### Knowledge
- [ ] All four foundation lessons read
- [ ] Quiz scores ≥ 7/10 on at least 3 of 4 lessons
- [ ] Constitution v1.1 read
- [ ] Glossary read in full pass
- [ ] Cert Framework read

#### Practical
- [ ] 8 recorded Weekly Vital Reads
- [ ] ≥ 3 Loop entries with verification dates
- [ ] ≥ 1 Loop entry fully closed (including failures)
- [ ] ≥ 1 Customer Economics analysis
- [ ] ≥ 1 Pillar 8 verification pause
- [ ] ≥ 1 verified positive business outcome

#### Role-play
- [ ] Passed ≥ 2 of 3 Scenario A role-plays
- [ ] Named trap avoided in each pass

#### Evidence pack
- [ ] All universal items present
- [ ] All Operator I tier items present
- [ ] ≥ 1 honest named failure documented
- [ ] Pioneer Test self-application completed

#### Doctrine
- [ ] Can recite Pioneer Test (4 questions)
- [ ] Can name all 6 Vitals
- [ ] Can describe Loop (6 stages, in order)
- [ ] Can explain Pillar 8 discipline
- [ ] Can describe AI Breadth/Judgment/Truth division

### Operator II Gate (stretch)

#### Cumulative
- [ ] All Operator I gates passed at higher bar
- [ ] 8+ QBRs delivered, rubric pass on ≥ 6
- [ ] 5+ Customer Economics analyses
- [ ] 1+ Self-Sustaining Loop verified
- [ ] 1+ churn-risk save documented
- [ ] 1+ expansion deal closed
- [ ] 1+ Pillar 8 Discontinuity handled
- [ ] All Scenario A passed
- [ ] ≥ 2 stretches passed

#### Business outcomes (12-month trailing)
- [ ] NRR ≥ 105%
- [ ] GRR ≥ 90%
- [ ] Customer NPS ≥ 50
- [ ] Health average ≥ 75

#### Portfolio artifacts
- [ ] Implementation case study
- [ ] Executive briefing recorded
- [ ] Difficult conversation documented
- [ ] Customer health model in active use
